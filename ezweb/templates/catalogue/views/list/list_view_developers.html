Nuevo gadget a partir de feed RSS
¿No está tu feed RSS favorito en el catálogo de EzWeb? Tranquilo, hay una solución.
Simplemente introduciendo la URL del feed en un asistente, puedes crear automáticamente tu propio gadget lector de noticias. Además, todo el mundo podrá usar tu gadget.
¡Así de fácil es crear un gadget en EzWeb!
Nuevo gadget a partir de sitio web
Otra forma sencilla de crear gadgets en EzWeb es a partir de una URL de un sitio web.
Usando un simple asistente, podrás crear automáticamente los gadgets de tus sitios web favoritos. ¡Pruébalo, tardas menos de 1 minuto!
Nuevo gadget a partir de template
La creación de este tipo de gadgets no puede automatizarse (ya que cada uno puede tratar temas diferentes) y requiere conocimientos de programación. Está, por tanto, solo recomendado para usuarios avanzados.
Un template es un fichero XML escrito por el desarrollador del gadget, expresando la funcionalidad que cada gadget usa de la Plataforma EzWeb. Para crear un nuevo gadget, su debe escribir la URL del template en el siguiente campo de texto y hacer click en el botón adjunto.
Nuevo gadget a partir de paquete
Para facilitar la tarea de los desarrolladores de gadgets, se ha creado un entorno de desarrollo (IDE) de gadgets. Esta herramienta genera un único fichero comprimido con todos los ficheros que componen el gadget.
Para agregar un gadget comprimido en un paquete, solo hay que localizar el fichero del gadget en el siguiente campo de texto y hacer click botón adjunto.
